A short wall-mounted rack in a coat closet, replacing Google mesh Wi-Fi with a wired UniFi network - a UXG Fiber gateway, one U7 Pro XG access point, and Flex 2.5G switches feeding both offices. A Minisforum mini PC running Proxmox hosts the UniFi Network controller, since the gateway doesn't run it on its own.
